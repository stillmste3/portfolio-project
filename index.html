Weather App

The weather app is a user-friendly web application built using HTML, CSS and JavaScript. It provides real time weather information for locations all around the world. The app alllows users to enter the name of a city or use their device's location using geolocation to retrieve weather data for their current location. The app's interface is designed using HTML and CSS to provide visually appealing and intuitive user experience. It displays essential weather information such as temperature, humidity, wind speed and weather conditions for the selected location. JavaScript is used to fetch weather data from a weather API, parse the retrieved data, and dynamically update the app's interface.
